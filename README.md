## 🏗️ 核心方法论 (Core Frameworks)

### 🥇 The 5% Rule (5% 法则)
> **“95% 的 SEO 工作是低效的，只有 5% 真正驱动了 95% 的结果。”**
*   **策略**：通过严格的归因分析，识别出那些真正产生转化的“5% 页面”或“5% 动作”。
*   **执行**：无情地砍掉无意义的路线图，集中所有资源重复和放大这 5% 的高产出路径。

### 🤖 Answer Engine Optimization (AEO)
*   **核心理念**：搜索的未来不再是“蓝色链接”，而是 AI 的直接回答。
*   **动作**：
    *   **Content Gaps**：针对 AI 经常提及的问题进行针对性内容补强。
    *   **Citation Strategy**：让你的品牌成为 AI 首选的、最权威的参考文献。
    *   **Format for LLMs**：使用更容易被语言模型理解的数据结构和语义标记。
