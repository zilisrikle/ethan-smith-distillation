## ⚡ 核心方法论预览

### 🥇 5% 法则
> 95% 的 SEO 工作是低效的。用数据找到真正驱动 95% 结果的那 5%，然后只做那 5%。

### 🤖 AEO / GEO（答案引擎优化）
> 搜索的未来是 AI 引用，不是蓝色链接。核心动作：Citation Engineering + 内容可提取性 + Answer Share of Voice 监测。

### 🏆 Topical Authority
> 以 Topic（话题）替代 Keyword（关键词）。一篇 Pillar 文章覆盖整个 Topic 下 100-500 个相关词。
